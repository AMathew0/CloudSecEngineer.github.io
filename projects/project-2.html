TechVerse Infrastructure Lab

Description: A real-world hybrid IT infrastructure simulation lab designed for educational organizations or small to medium enterprises. This project demonstrates secure network architecture, hybrid cloud integration, endpoint security, automation, and documentation using a blend of open-source and enterprise-grade technologies.

Project Objectives:

  • Design and implement a scalable hybrid IT infrastructure model.
  • Demonstrate secure cloud and on-prem integration using best practices.
  • Utilize open-source tools for cost-effective IT service management.
  • Simulate enterprise-level networking, monitoring, and automation workflows.

Project Outcome:

  • Deployed end-to-end hybrid IT environment for education use case.
  • Integrated Microsoft 365, AWS, and Linux services for identity, storage, and compute.
  • Implemented monitoring, disaster recovery, and endpoint security solutions.
  • Developed documentation, backup strategy, and ITSM workflow.

Key Skills Gained:

  • Hybrid network design and VLAN segmentation
  • Windows Server AD DS, DNS, DHCP configuration
  • Linux server setup, Proxmox, and TrueNAS storage solutions
  • AWS EC2, S3, Route 53, and Azure AD integration
  • Automation with Ansible, Terraform, and scripting (Bash & PowerShell)
  • Monitoring and logging with Zabbix, Grafana, and Graylog
  • IT documentation, service desk, and backup strategies

Technologies Used:

  • AWS (EC2, S3, RDS, Route53)
  • Azure AD, Intune, M365, Defender
  • Ubuntu 22.04 LTS, Proxmox VE
  • Windows Server 2019 (AD DS, DNS, DHCP)
  • FortiGate Firewall, UniFi Wi-Fi, Cisco Switches
  • PostgreSQL, TrueNAS CORE (ZFS)
  • BitLocker, LUKS, Defender for Endpoint
  • Ansible, Terraform, PowerShell, Bash
  • Zabbix, Grafana, Graylog, GLPI
  • Intune, WSUS, PDQ Deploy
